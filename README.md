# MS Design System

### Version

> 1.0

## Description

A design system UI offers reusable components and guidelines for consistent, efficient design across digital products.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

PORT=3200
